What DP does

Enumerates ~2,400 X-to-move states once. For each, applies the Bellman backup using a uniform-random opponent model. Converges in ~4 sweeps.

V(s) ← max_a Σ P(s′|s,a) [r + γ V(s′)]

What MC does

Plays one full self-play episode at a time. Records visited states. At the end, every first-visit state gets V(s) ← V(s) + (1/n) · [G − V(s)]. Takes ~50k–100k episodes to settle.

What TD does

Same self-play loop, but updates fire after every move, not at episode end. Uses bootstrapping: V(s) ← V(s) + α [r + γ V(s′) − V(s)]. Same answer, ~10× fewer games.
